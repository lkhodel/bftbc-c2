## Usage

The app runs from the root directory and has no options. 

~~~
$ python app.py -h

NAME
    app.py

SYNOPSIS
    app.py COMMAND

COMMANDS
    COMMAND is one of the following:

     run
       The main function for running the script.

     test
       Test with hardcoded parameters.
~~~
